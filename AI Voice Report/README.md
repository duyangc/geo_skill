# GEO AI 声量诊断报告生成技能 (geo_skill)

品牌 AI 搜索声量分析报告（GEO 诊断报告）的 WorkBuddy 技能包。基于标准红色配色方案（#D92B2B），一键生成专业级 HTML 格式 GEO 诊断报告。

## 功能概述

- 自动解析 AI 平台（千问/豆包/DeepSeek 等）的测试结果数据
- 生成 6 大章节的专业诊断报告：核心声量概览 → 认知现状 → 分类分析 → 竞品对比 → 问题诊断 → 行动路径
- 支持品牌类与非品牌类两种报告模式
- 内置 8 维质量自检体系，减少人工审校成本

## 安装方法

### 方式一：手动安装（推荐）

1. 下载本仓库的所有文件
2. 将 `geo_skill` 文件夹复制到 WorkBuddy 的技能目录：
   ```bash
   # macOS/Linux
   cp -r geo_skill ~/.workbuddy/skills/

   # Windows
   # 将 geo_skill 文件夹复制到 %USERPROFILE%\.workbuddy\skills\
   ```
3. 重启 WorkBuddy 客户端
4. 在对话中触发：`使用 ai-voice-report 技能生成报告`

### 方式二：从 WorkBuddy 市场安装

（如有发布到市场）在 WorkBuddy 对话中输入：`/install geo_skill`

## 使用流程

1. **准备数据**：收集各 AI 平台的测试结果（测试问题 + AI 回答原文）
2. **加载技能**：在 WorkBuddy 中加载 `ai-voice-report` 技能
3. **输入指令**：告知技能目标品牌名称、测试平台、测试结果文件路径
4. **生成报告**：技能会自动完成品牌调研 → 数据统计 → 报告生成 → 质量自检的全流程
5. **交付报告**：输出 `.html` 格式的完整报告

## 模板关键特性

- **纯 HTML+CSS 实现**：不依赖外部 JS 库，打开即用
- **自适应列数**：`platform-grid` 支持 cols-2/cols-3/cols-4
- **SVG 环形评分图**：自动根据分数计算 stroke-dashoffset
- **三阶段竖向堆叠**：行动路径默认纵向排列，适配长文本
- **内置自检清单**：每章节末尾有 HTML 注释格式的自检项
- **占位符标记**：使用 `==PLACEHOLDER==` 格式，醒目不易遗漏

## 注意事项

- 本技能生成的数据驱动型诊断报告，**所有数据必须有实测来源**，不得虚构
- 报告中的 GEO 优化建议侧重**内容撰写与媒体投放**，不含服务商内部运营工作
- 品牌名称、平台名称、竞品名称等需根据实际测试内容替换
- 三阶段行动路径的任务描述中**不得出现具体时间承诺或具体媒体/KOL 名称**
